  - 그리고 메모리의 작동 원리나 과정에 관련해서는 가상 메모리의 개념이 빠질 수 없는 것 같다.
  - 그와 관련된 공간 스왑, 페이지 폴트, 페이지 교체 알고리즘 등에 대해 깊은 공부를 해야 할 것 같다고 생각했다.
